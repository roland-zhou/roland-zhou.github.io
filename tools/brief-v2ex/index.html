Brief V2EX

Settings

Create one at v2ex.com/settings/tokens. Without a token we fall back to the legacy v1 API, which is public but flaky and rate-limited.
V2EX doesn't send CORS headers, so direct browser requests are blocked. A proxy URL will be prefixed before each V2EX URL (which is appended URL-encoded). Leave empty to call V2EX directly. Quick start: https://corsproxy.io/?url= or https://api.allorigins.win/raw?url=. Recommended: deploy your own free Cloudflare Worker — see cloudflare-worker.js for the script and 5-step setup. Free tier covers 100k requests/day, and only V2EX hosts are allowed so it can't be abused as an open proxy. Warning: public proxies see all traffic, including any V2EX token. Delete your token before testing through one, or run your own proxy.
